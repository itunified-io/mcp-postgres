# Skills Reference — mcp-postgres

## Overview

The OSS `mcp-postgres` server provides 27 tools for PostgreSQL operations but does not include interactive workflow skills. Skills are available in [mcp-postgres-enterprise](https://github.com/itunified-io/mcp-postgres-enterprise).

## Available Skills

None — the OSS package focuses on individual tools. For orchestrated workflows (health checks, compliance audits, migration planning, etc.), use the enterprise edition which provides 30 skills.

## Tool Categories

The 27 OSS tools are organized into these categories:

| Category | Tools | Description |
|----------|-------|-------------|
| Connection | 5 | Multi-database connection management |
| Query | 3 | SQL execution and explain plans |
| Schema | 9 | Schema introspection (tables, indexes, views, functions) |
| CRUD | 4 | Data manipulation (insert, update, delete, upsert) |
| Database | 2 | Database size and table sizes |
| Server | 4 | Version, settings, config reload, uptime |

See the [main README](../README.md) for the complete tool reference.
